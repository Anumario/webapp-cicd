Résumé du Projet

Ce projet consiste à mettre en place un pipeline CI/CD complet permettant l’automatisation de l’intégration, des tests et du déploiement d’une application web. Le pipeline utilise Jenkins pour l’orchestration, Docker pour la conteneurisation, Kubernetes pour le déploiement, et GitHub / GitHub Container Registry pour la gestion du code source et des images.
